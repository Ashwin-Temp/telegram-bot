# YouTube Downloader Telegram Bot

## STEP 1: Setup
1. Install Python 3.10 or newer
2. Open terminal and run:
   pip install -r requirements.txt

## STEP 2: Fill Credentials
Edit `.env` file and fill in:
- API_ID
- API_HASH
- BOT_TOKEN
- CHANNEL_ID (use -100 format for private, or username for public)

## STEP 3: Run the bot
python bot.py

## Optional:
- All videos are saved in "downloads/" before sending.
- Only users who join the Telegram channel can use the bot.
